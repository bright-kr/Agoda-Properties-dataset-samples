## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | The URL linking directly to the property listing on Agoda. |
| `hotel_id` | A unique identifier for each hotel property listed on Agoda. |
| `title` | The name or title of the hotel property. |
| `location` | The full address of the hotel, including street, city, and postal code. |
| `country` | The country where the hotel is located. |
| `city` | The city in which the hotel is located. |
| `metro_railway_access` | Indicates whether the hotel is accessible via metro or railway; true or false. |
| `images` | A list of URLs pointing to images of the hotel property. |
| `number_of_reviews` | The total number of reviews the hotel has received. |
| `review_score` | The average review score given by guests, typically ranging from 0 to 10. |
| `description` | A detailed description of the hotel property, including features and services. |
| `property_highlights` | Key features and highlights of the property that make it stand out. |
| `most_popular_facilities` | Facilities that are frequently highlighted as popular by guests. |
| `availability` | Indicates the current booking availability of the hotel. |
| `reviews_scores` | A breakdown of review scores across different categories like cleanliness, service, etc. |
| `top_reviews` | Excerpts from the most helpful or highest-rated guest reviews. |
| `property_information` | General information about the property, including policies and operational details. |
| `manager_language_spoken` | Languages spoken by the hotel management or staff. |
| `property_surroundings` | Information about the surrounding area of the property, including nearby attractions. |
| `popular_facilities` | Facilities commonly used or appreciated by guests during their stay. |
| `fine_print` | Important terms and conditions or additional policies for staying at the hotel. |
| `latitude` | The geographical latitude of the hotel location. |
| `longitude` | The geographical longitude of the hotel location. |
| `location_rating_score` | A score reflecting guests' satisfaction with the hotel's location. |
| `good_to_know` | Additional useful information about the hotel that guests should be aware of. |
| `rooms_available` | Details about the types and number of rooms currently available at the hotel. |
| `faq` | Frequently asked questions about the hotel, including booking and amenities. |
| `amenities_and_facilities` | A comprehensive list of amenities and facilities offered by the hotel. |
| `walkable_places` | Nearby places of interest that can be reached on foot from the hotel. |
| `location_details` | In-depth details about the hotel's location, including accessibility and neighborhood. |
| `whats_nearby` | Attractions, landmarks, and facilities located near the hotel. |
| `some_helpful_facts` | Interesting or helpful facts about the hotel or its location that may benefit guests. |
